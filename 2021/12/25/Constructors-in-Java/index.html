Types of Constructors

There are two types of constructors in Java: No-argument constructor and Parameterized Constructor.

No-argument constructor

1.Explanation:
“A constructor that has no parameter is known as the default constructor“ (GeeksforGeeks, 2021);

“If we don’t define a constructor in a class, then the compiler creates default constructor(with no arguments) for the class” (GeeksforGeeks, 2021);

“If we write a constructor with arguments or no-arguments then the compiler does not create a default constructor” (GeeksforGeeks, 2021)；

“Default constructor initializes the the instance variables with default values, like 0, 0.0, false, null, etc, which is depending on the variable’s type” (GeeksforGeeks, 2021).

The return type of a constructor

1.Explanation:
“A constructor do not have any return type, not even void. There are no ‘return value’ statements in the constructor, but the constructor returns the current class instance. We can write ‘return’ inside a constructor” (GeeksforGeeks, 2021).
